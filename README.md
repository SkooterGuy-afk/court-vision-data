# court-vision-data

## Project Overview
This repository hosts a self-directed data analytics project exploring the intersections of sports statistics, predictive mathematical modeling, and data engineering. By analyzing historical basketball data, the project builds algorithms to isolate key victory variables and calculate statistical probabilities for game outcomes. 

I launched this portfolio ahead of my junior year of high school to actively bridge my business and finance interests—developed through DECA—into practical programming environments. My long-term goal is to leverage these data skill sets to pursue a dual concentration in Finance and Business Analytics at the University of Colorado Boulder (Leeds School of Business).

## Key Objectives
* **Data Engineering:** Build automated data pipelines to extract and clean multi-season historical basketball metrics using Python.
* **Feature Optimization:** Implement regression math to weight performance variables (such as turnover metrics vs. effective field goal percentages).
* **Predictive Modeling:** Generate win/loss probability distributions, simulating the foundational risk-assessment principles used on quantitative finance desks.

## Core Tech Stack
* **Language:** Python
* **Data Wrangling:** Pandas, NumPy
* **Mathematical Modeling:** Scikit-Learn
* **IDE:** Jupyter Notebook / Python 3
